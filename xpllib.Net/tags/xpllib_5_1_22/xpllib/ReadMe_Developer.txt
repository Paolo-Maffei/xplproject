How to create a new release;
 - check the assembly info on the project tab
 - the build number is going to be the build number shown +1 (if auto increment is used)
 - update the readme file, with changelog for buildnumber xxx
 - set type to "release" and Rebuild solution
 - exit Visual Studio, saving any changes
 - package the xpllib.dll and the xpllib.xml files in the bin/release folder
