calculateevent (self, ev)
Calculates the expire field of an event. It will calculate the next time, from the current expire value. To calculate from scratch, set expire to nil before calling this method.

Parameters:

  • self:
  • ev: event to calculate the expire time for. Either a single or repeating event.

Return value:

    the expire property of ev will be set to the new expire time.
